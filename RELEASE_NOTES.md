## 🚀 快速开始

### 1. 环境要求
- Python 3.7+
- MySQL 8.0+
- 依赖包：mysql-connector-python, tabulate

### 2. 安装步骤
```bash
# 1. 克隆项目
git clone https://github.com/9531lyj/bankManage.git
cd bankManage

# 2. 安装依赖
pip install -r requirements.txt

# 3. 配置数据库
# 确保MySQL服务运行，创建数据库'银行'

# 4. 启动GUI界面
python start_gui.py
# 或者
python BankSystemGUI.py
```
